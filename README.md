# Testes E2E - MyFarm

Projeto de testes automatizados E2E do MyFarm com Cypress

## Configurar projeto

```bash
# instalar dependencias
npm install
```

## Executar testes - CLI

```bash
# Allure reporter (local)
npm run test:allure

# Junit reporter (azure)
npm run test:azure
```

## Abrir interface do Cypress

```bash
# cypress runner
npm run cypress:open
```

## Abrir interface do Allure
